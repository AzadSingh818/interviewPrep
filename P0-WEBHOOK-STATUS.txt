│  YOUR API VERIFIES SIGNATURE ✓                                             │
│              ↓                                                               │
│  CHECKS FOR DUPLICATES ✓                                                   │
│              ↓                                                               │
│  PROCESSES EVENT (grants subscription/unlock)                              │
│              ↓                                                               │
│  LOGS TO DATABASE & SENTRY                                                 │
│              ↓                                                               │
│  USER SEES SUCCESS                                                          │
│       (Subscription/Feature active)                                        │
│              ↓                                                               │
│  EMAIL SENT TO USER                                                         │
