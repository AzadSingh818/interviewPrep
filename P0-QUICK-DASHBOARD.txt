STEP 1: Configure Razorpay Webhook (Do This First - 15 min)

File: P0-WEBHOOK-EXECUTABLE.txt (complete step-by-step guide)

Quick summary:
  1. Open: https://dashboard.razorpay.com
  2. Login with your account
  3. Navigate: Settings → Webhooks
  4. Click: "+ Add New Webhook"
  5. Fill:
     • URL: https://your-domain.com/api/webhooks/razorpay
     • Events: payment.captured, payment.failed, order.paid
  6. Click: Save
  7. Copy: Webhook secret (save for next step)
  8. Test: Click "Test Webhook" → Get 200 OK

Result: ✅ Webhook created, secret copied

║                    🔧 TROUBLESHOOTING QUICK GUIDE                            ║
╚═══════════════════════════════════════════════════════════════════════════════╝

Problem: Webhook test returns error
Solution:
  • Wait 30 seconds, try again
  • Check URL has no typos
  • Verify HTTPS (not HTTP)
  • Proceed anyway (will work after deploy)

Problem: Secrets not working after adding to Vercel
Solution:
  • Wait 5 minutes for redeploy
  • Check variable names (case-sensitive)
  • Verify no extra spaces in values
  • Check in Vercel logs for errors

Problem: Payment fails
Solution:
  • Check webhook secret is correct (matches Razorpay)
  • Verify Sentry DSN is valid
  • Check Vercel logs for errors
  • Verify test mode vs live mode

Problem: Emails not sending
Solution:
  • Check email job queue implemented
  • Verify SMTP credentials
  • Check Sentry for email errors
  • Check logs for queue status
